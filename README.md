## 🚀 About

**Bharat Biz OS** is an AI-powered SaaS platform that helps Indian MSMEs manage CRM, GST billing, inventory, accounting, analytics, and business operations from a single dashboard.

Designed for businesses that need an affordable, modern alternative to managing multiple disconnected tools.

## 📂 Installation

Clone the repository

```bash
git clone https://github.com/razer0x1/bharat-biz-os.git
```

Go into the project

```bash
cd bharat-biz-os
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
VITE_GEMINI_API_KEY=your_gemini_api_key
```

Run the project

```bash
npm run dev
```
